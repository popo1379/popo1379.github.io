2017-03-23 • 标签: CS基础

一 数学基础：

定义： 如果T(N)增长率<=f(N)增长率，则T(N)=O(f(N)) (大O标记)
如果T(N)增长率>=f(N)增长率，则T(N)=Ω(f(N))
如果T(N)增长率=f(N)增长率，则T(N)=Θ(f(N))
如果T(N)增长率<f(N)增长率，则T(N)=o(f(N))

法则1：如果T1(N)=O(f(N))且T2(N)=O(g(N)) 则

T1(N)+T2(N)=O(f(N)+g(N))
T1(N)T2(N)=O(f(N)g(N))

法则2： 如果T(N)是一个K次多项式，T(N)=Θ(N^K)

法则3： 对于任意的常数k,(logN)^k = O(N)

法则4： 常数项或低阶项不要放入O（）中。
